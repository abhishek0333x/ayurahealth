# SEO Skill

This skill provides comprehensive SEO coverage including technical audits, content optimization, schema markup, and strategic planning.

## Sub-Skills

**1. /seo audit** — Full site technical audit covering crawlability, indexability, performance, mobile-friendliness, and security.

**2. /seo page** — Single page deep analysis including on-page SEO, keyword optimization, readability, and internal linking.

**3. /seo content** — E-E-A-T (Experience, Expertise, Authoritativeness, Trustworthiness) quality scoring for content.

**4. /seo schema** — JSON-LD structured data generation for rich snippets, breadcrumbs, FAQ, product reviews, etc.

**5. /seo geo** — AI Overviews and geographic optimization for local search and global reach.

**6. /seo plan** — Strategic SEO planning including keyword research, competitor analysis, and roadmap.

**7. /seo competitor** — Gap analysis comparing your site to competitors' SEO strategies.

## SEO Health Score (0-100)

The SEO Health Score evaluates 7 weighted categories:

1. **Technical SEO (25%)** — Crawlability, indexability, site speed, mobile-friendliness, security
2. **On-Page SEO (20%)** — Title tags, meta descriptions, headers, keyword usage, content quality
3. **Content Quality (20%)** — E-E-A-T, readability, depth, freshness, uniqueness
4. **Link Profile (15%)** — Backlinks, internal links, anchor text, domain authority
5. **User Experience (10%)** — Click-through rate, bounce rate, time on page, conversion rate
6. **Schema Markup (5%)** — Structured data coverage, rich snippets, breadcrumbs
7. **Mobile Optimization (5%)** — Responsive design, mobile speed, touch-friendly

## Industry Detection

The skill automatically detects your industry and applies industry-specific SEO rules:

**SaaS:** Focus on product pages, pricing, case studies, technical documentation
**E-commerce:** Focus on product pages, category pages, user reviews, rich snippets
**Local:** Focus on Google My Business, local citations, location pages
**Publishers:** Focus on content freshness, topical authority, E-A-T signals
**Healthcare:** Focus on YMYL (Your Money Your Life) guidelines, medical accuracy, citations
**Wellness:** Focus on holistic health content, practitioner credentials, user testimonials

## SEO Audit Checklist

**Technical SEO:**
- [ ] Robots.txt properly configured
- [ ] Sitemap.xml submitted to search engines
- [ ] SSL certificate installed (HTTPS)
- [ ] Mobile-friendly design
- [ ] Page speed > 3 seconds
- [ ] No duplicate content
- [ ] Proper URL structure
- [ ] Internal linking strategy

**On-Page SEO:**
- [ ] Title tags (50-60 characters, keyword included)
- [ ] Meta descriptions (150-160 characters, compelling)
- [ ] H1 tags (one per page, keyword included)
- [ ] H2/H3 tags (proper hierarchy)
- [ ] Alt text on all images
- [ ] Keyword usage (1-2% density)
- [ ] Internal links (3-5 per page)

**Content Quality:**
- [ ] Original, unique content
- [ ] Comprehensive coverage of topic
- [ ] E-E-A-T signals present
- [ ] Readability score > 60
- [ ] Content length > 1000 words (for competitive keywords)
- [ ] Regular updates and freshness

**Schema Markup:**
- [ ] Organization schema
- [ ] Product schema (if applicable)
- [ ] Review schema (if applicable)
- [ ] FAQ schema (if applicable)
- [ ] Breadcrumb schema
- [ ] Article schema (for blog posts)
