# UI/UX Pro Max Skill

This skill applies design intelligence to automatically select the right style, palette, and typography for your product, then generates a complete design system.

## Design System Selection Process

1. **Analyze Product** — Understand the core value proposition, primary use case, and user journey
2. **Identify Audience** — Determine demographics, technical proficiency, and design preferences
3. **Select Style** — Choose from 67 design styles based on product-audience fit
4. **Choose Palette** — Select from 161 color palettes that reinforce your style
5. **Pick Fonts** — Select from 57 font pairings that enhance readability and brand
6. **Generate Tokens** — Create design system tokens for colors, typography, spacing, shadows

## Design Styles (Sample)

**Luxury:** Premium, refined, high-end products (jewelry, luxury travel, wealth management)

## UX Rules (99 Total)

**Spacing:** Use consistent 8px grid, multiples of 8 for all spacing
**Touch Targets:** Minimum 44px for mobile, 32px for desktop
**Contrast:** WCAG AA minimum (4.5:1 for text, 3:1 for graphics)
**Responsive:** Mobile-first design, test at 320px, 768px, 1024px, 1440px
**Loading:** Skeleton screens, never spinners
**Feedback:** Every action should have visual feedback (hover, active, disabled)
**Accessibility:** Keyboard navigation, focus states, alt text, ARIA labels

## Pre-Delivery Checklist

- [ ] Contrast ratios meet WCAG AA (text 4.5:1, graphics 3:1)
- [ ] Touch targets minimum 44px on mobile
- [ ] Responsive design tested at 320px, 768px, 1024px, 1440px
- [ ] All interactive elements have hover/focus/active states
- [ ] Loading states use skeleton screens
- [ ] Keyboard navigation works throughout app
- [ ] Focus states visible and clear
- [ ] Alt text on all images
- [ ] ARIA labels on custom components
- [ ] Color not used as sole means of conveying information
